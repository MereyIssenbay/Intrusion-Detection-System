# Intrusion Detection System (IDS)
Использование YOLO для обнаружения объектов

Библиотека OpenCV для захвата видео, рисования полигонов, отображения текста и обработки кадров.

Использование Shapely для работы с геометрией для проверки пересекает ли объект ограниченную зону.

Matplotlib для визуализации. Визуализаци включает отображение ограниченных зон и объектов, находящихся в этих зонах.


# Запуск
Установите все нужные библиотеки из requirements.txt

```python
pip install -r requirements.txt
```

Запустите все модели по очереди. Укажите точки полигона.
